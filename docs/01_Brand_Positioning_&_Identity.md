# Purpose
This document defines the core brand identity and positioning rules.
All outputs must align with this.

# Youtube Channel vibe
**Think:**

- Ali Abdaal's productivity + Fireship's tech humor + Casey Neistat's raw vlogs
- Clean but not corporate
- Energetic but authentic
- Educational but entertaining

**Visual Style:**

- Simple thumbnails (your face + bold text)
- Consistent color scheme (2-3 colors)
- Good lighting (natural or cheap ring light)
- Decent audio (phone + $20 mic fine initially)

**Tone:**

- Humble but confident
- "Here's what I'm learning" not "I'm an expert"
- Encouraging to others
- Honest about failures

# Youtube niche statement
**"I help ambitious students in Bangladesh build tech skills and earn online by documenting my journey from 15 to financial freedom through web development, video editing, and automation."**

**Target Audience:**
- 14-22 year olds in Bangladesh/South Asia
- Want to escape traditional path
- Interested in tech/freelancing
- Need practical, relatable guidance

**Your Promise:**
- Real results, not hype
- Beginner-friendly
- Culturally relevant
- Achievable with limited resources
